SquidgyCMS is a Flat-file Content Management System. This means that is does not require a database to run - all you need to do is copy the source files into a your web directory (frequently called public_html on *nix based servers) and you have a functional website. Configuration is done through two vectors - most seetings are availabe through the administration interface, while the more security sensitive settings and some of those you are less likely to change once set are set in a settings file.
